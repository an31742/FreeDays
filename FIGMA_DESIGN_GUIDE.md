# Figma设计指南：节假日倒计时小程序界面

## 整体布局

### 页面结构
- 顶部标题区域
- 节假日卡片列表区域
- 页面间距：上下左右各20rpx（约10px）

### 背景设计
- 背景色：线性渐变
- 起始色：#f5f7fa
- 结束色：#c3cfe2
- 渐变角度：135度

## 顶部标题区域

### 标题文本样式
- 文本内容："节假日倒计时"
- 字体大小：40rpx（约20px）
- 字体粗细：bold（700）
- 颜色：#2c3e50
- 文字阴影：2rpx 2rpx 4rpx rgba(0,0,0,0.1)
- 对齐方式：居中

## 节假日卡片设计

### 卡片整体样式
- 背景色：白色 (#FFFFFF)
- 圆角：16rpx（约8px）
- 阴影：
  - 垂直偏移：8rpx
  - 模糊半径：32rpx
  - 颜色：rgba(0, 0, 0, 0.08)
- 卡片间距：24rpx（约12px）

### 卡片悬停效果
- 垂直偏移：16rpx
- 模糊半径：48rpx
- 颜色：rgba(0, 0, 0, 0.12)
- Y轴位移：-6rpx

### 左侧颜色标识
- 宽度：4rpx
- 颜色：按节日区分
  - 春节：#FF6666
  - 清明节：#66CC66
  - 劳动节：#FFCC33
  - 端午节：#3399FF
  - 中秋节：#9966CC
  - 国庆节：#9966CC

### 卡片内容区域
- 内边距：32rpx（约16px）

## 节假日标题行

### 节日名称
- 字体大小：36rpx（约18px）
- 字体粗细：bold（700）
- 颜色：#2c3e50
- 位置：左侧

### 剩余天数标签
- 字体大小：32rpx（约16px）
- 字体粗细：bold（700）
- 内边距：8rpx 20rpx
- 圆角：32rpx
- 默认背景：#f8f9fa
- 默认文字颜色：#495057

### 紧急状态（剩余7天以内）
- 背景：#ffeaa7
- 文字颜色：#d68910

### 已过期状态
- 背景：#e9ecef
- 文字颜色：#6c757d

## 描述文本
- 字体大小：28rpx（约14px）
- 颜色：#6c757d
- 行高：1.5倍
- 底部边距：16rpx

## 日期信息
- 字体大小：26rpx（约13px）
- 颜色：#868e96
- 字体粗细：500
- 底部边距：24rpx

## 节日图片容器
- 宽度：100%
- 高度：200rpx（约100px）
- 圆角：12rpx
- 图片填充模式：aspectFill（保持比例填充）

## 动画效果
- 进入动画：fadeInUp
- 持续时间：0.6秒
- 时间函数：ease
- 不同卡片延迟出现：
  - 第1个：0.1秒
  - 第2个：0.2秒
  - 第3个：0.3秒
  - 第4个：0.4秒
  - 第5个：0.5秒
  - 第6个：0.6秒

## 组件尺寸参考
- 小程序屏幕宽度：375px（标准iPhone）
- 小程序屏幕高度：812px
- 卡片宽度：接近屏幕宽度减去边距
